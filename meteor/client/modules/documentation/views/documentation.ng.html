Overview

The purpose of this document is to show you the easiest way to get started building your own web apps! The document outlines the way the web app is written. Enjoy and happy coding!

Meteor's documentation contains many more details of how the system works as full-stack JavaScript.


Getting Started

Tips

  • Do not start from scratch! Use existing modules, code, etc. to jump-start your ideas.
  • Use Firefox Inspector, Debugger, Network, and Console tools to debug your app. (Access via "Right-Click->Inspect Element")
  • Most errors you come across can be found on Google. If not, you may be doing something incorrect.
  • CSS Flexbox can be a bit tricky. Look at how core.html is structured with all the other modules. Flexbox is very sensitive with the nested use of flex .

Menus

The main Menu controller is located at: client/lib/modules/menu/MenuCtrl.ng.js. Modify topbar notifications, messages, etc. and Profile card (above sidebar) contents.

Sidebar

To modify Sidebar contents:

  1. Edit client/lib/modules/menu/SideMenuService.ng.js
  2. Add routes to sidebar links client/lib/modules/core/routes.ng.js


Top Header

To modify the top navigation:

  1. Edit client/lib/modules/menu/views/includes/header_toolbar.ng.html
  2. Add navigation routes in client/lib/modules/core/routes.ng.js. Don't forget to include all controllers, directives, and templates!


Profile Card

To modify the profile card:

  1. Edit client/lib/modules/menu/views/includes/profile_card.ng.html
  2. Edit client/lib/modules/menu/MenuCtrl.ng.js

The parent/child relationship shown above is made possible through the use of <div ui-view> </div> . In other words, core.html is the "base" template that builds all other pages through html-includes and partials (ui-view) [router].


Themes

To change the colors of this theme you need to edit the following files:
  • client/lib/theme/mdStyleColor.js - contains Material theme colors for primary, accent, and warn colors.
  • client/styles/material-colors/md-theme.less - contains LESS variables (essentially CSS) for primary, accent, success, info, error, and warn colors.
  • client/modules/dashboard/controllers/DashboardCtrl.js - colorFunction() contains color definitions for the chart.

Seed Project

To jump start a new application, use the "seed project" (meteor-seed) as a skeleton for the base of your project. Meteor is a great choice to pick as it streamlines the development of both backend and frontend systems using all JavaScript, with many systems already developed such as authentication.

The seed project includes the following features:
  • Authentication
  • Dashboard
  • Sidebar, top toolbar, & Profile
  • Google Material design
  • Font Awesome & Material Icons
  • Chart libraries
    • Material Charts
    • nvD3
    • ChartJS.org
