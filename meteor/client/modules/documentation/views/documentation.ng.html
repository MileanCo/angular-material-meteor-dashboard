Overview

The purpose of this document is to show you the easiest way to get started building your own web apps! The document outlines the way the web app is written. Enjoy and happy coding!

Meteor's documentation contains many more details of how the system works as full-stack JavaScript.

Getting Started

Tips

  • Do not start from scratch! Use existing modules, code, etc. to jump-start your ideas.
  • Use Firefox Inspector, Debugger, Network, and Console tools to debug your app. (Access via "Right-Click->Inspect Element")
  • Most errors you come across can be found on Google. If not, you may be doing something completely incorrect
